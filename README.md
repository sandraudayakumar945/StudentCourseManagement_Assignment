# Student Course Management System

This project is an ASP.NET MVC web application developed as part of an academic assignment.  
The system is used to manage students, courses, and the mapping between students and courses.

## Project Description
The Student Course Management System allows administrators to:
- Manage student details
- Manage course details
- Map students to courses
- View student–course relationships

This project follows the MVC (Model–View–Controller) architecture.

## Technologies Used
- ASP.NET MVC
- C#
- SQL Server
- Entity Framework
- HTML, CSS
- Bootstrap

## Features
- Add, edit, delete students
- Add and manage courses
- Assign courses to students
- View mapped student–course data

## How to Run the Project
1. Open the project using Visual Studio
2. Update the database connection string in `appsettings.json`
3. Restore NuGet packages
4. Run the project using IIS Express
5. Access the application through the browser

## Database
- SQL Server is used as the backend database
- Entity Framework is used for data access
- Database configuration is available in `appsettings.json`
